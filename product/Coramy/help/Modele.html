Modèle

 

Le module Modèle contient tous les modèles.

Il y a deux méthodes pour se rendre dans le module Modèle :
     - à partir de la page d'accueil, cliquer sur modèles (cf 1)
     - quelque soit l'endroit où l'on se trouve, sélectionner dans la palette des modules (cf 2) le module Modèle
     et cliquer sur le bouton palette.
