Créer un nouveau modèle :

Il y a deux méthodes pour afficher un formulaire vierge :
     - Dans le menu action, choisir "Add Modele"et cliquer sur le bouton, ou
     - A partir du module Modele, cliquer directement sur l'icone d'un nouveau document

Dans ce formulaire vierge, nommer le modèle selon la nouvelle nomenclature
Remplir la référence client (si ellle existe)
Indiquer la Collection et l'EIP (Espace Imaginaire Produit)
Remplir la description
Dans Accesoires, il est possible d'annoter les transferts ou autre composant
Ne pas oublier de VALIDER
