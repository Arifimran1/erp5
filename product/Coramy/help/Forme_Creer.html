Forme

 

Créer une nouvelle forme :

Il y a deux méthodes pour afficher un formulaire vierge :
      - Dans le menu action, choisir "Add Forme"et cliquer sur le bouton, ou
      - A partir du module Forme, cliquer directement sur l'icone d'un nouveau document

Dans ce formulaire vierge, nommer la forme selon la nouvelle nomenclature
Remplir le référentiel forme
Indiquer la Collection et l'EIP (Espace Imaginaire Produit)
Remplir la description
Ne pas oublier de VALIDER

Dans le menu Action, ajouter un dessin technique (add dessin technique)
      Description du dessin (particulièrement s'il y en a plusieurs)
      VALIDER
Cliquer sur Parcourir, et choisir le dessin technique
